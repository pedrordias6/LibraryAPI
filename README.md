# LibraryAPI

A REST API for managing books in a library.

## Tech Stack

- Java 21
- Spring Boot 3
- Maven
- PostgreSQL
- Spring Data JPA
- Spring Validation
- Flyway
- Swagger / OpenAPI
- Docker
- JUnit 5
- Mockito
